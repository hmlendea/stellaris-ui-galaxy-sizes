# About

Mod for Stellaris that add more galaxy sizes to the game.

# Additions

 - Ludicrous (2000 stars)
 - Titanic (1800 stars)
 - Colossal (1600 stars)
 - Immense (1400 stars)
 - Gigantic (1200 stars)
 - Minuscule (100 stars)
